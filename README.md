# CyberShield AI

Competition-ready HTML, CSS and JavaScript website with real OpenAI vision analysis.

## Use

1. Open index.html or deploy the folder.
2. Click Connect AI.
3. Enter an OpenAI API key.
4. Paste text, upload a screenshot, or paste an image with Ctrl/Command + V.
5. Click Analyse with AI.

## Security

The key is stored only in sessionStorage. For a public production site, move the OpenAI request to a serverless function and keep the key in an environment variable.
